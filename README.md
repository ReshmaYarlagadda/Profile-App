# ApexPlanet Internship - Task 1

## Task Title:
Onboarding and Environment Setup + Mini Project: Profile App

## Objective:
This task focuses on setting up the development environment for Android app development and applying the basics through a mini project. The project demonstrates a simple **Profile App** created using Android Studio.

## Task Details:
- Installed and configured Android Studio, Java/Kotlin, Gradle, and necessary SDKs.
- Set up Git and created a GitHub repository.
- Learned basic Android components: Activities, Fragments, Layouts, and UI elements.
- Developed a mini project **“Profile App”** which includes:
  - User profile screen with name, email, and profile picture
  - Basic UI components like buttons, text fields, and images
  - Navigation between screens (if implemented)
- Screen recording of the project and environment setup prepared for LinkedIn.

## Deliverables:
1. Android Studio project files for the Profile App
2. README.md with task description
3. LinkedIn post with screen recording/demo of Profile App

## Skills Learned:
- Android Studio basics and project structure
- Java/Kotlin coding in Android
- Layouts, Activities, and basic UI design
- Git and GitHub version control
